# Financial-Engineering-Quant-101
This repository tracks my structured journey into **financial engineering** and **quantitative finance**, implemented through reproducible Jupyter notebooks.

## Goals

- Build a solid foundation in stochastic calculus, derivatives pricing, and risk.
- Implement models from scratch to understand assumptions and limitations.
- Develop a portfolio of clean, well-documented quant code.
